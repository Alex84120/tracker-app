📱 INSTALLATION DE TRACKER (Vite + React)

1. Crée un dépôt GitHub vide (ex: tracker-app)
2. Upload tous les fichiers ci-dessus, y compris le dossier /src
3. Va sur https://vercel.com/import
   ➜ Connecte ton GitHub
   ➜ Sélectionne ton dépôt
   ➜ Clique sur DEPLOY

4. Quand le site est en ligne, ouvre-le sur ton téléphone
   ➜ Clique sur “Ajouter à l’écran d’accueil”

✅ Ton app Tracker est maintenant installée !

Fonctionnalités incluses :
- Suivi muscu, whey, poids, écarts
- Remplacements, séances maison
- Intensité, rappels, dark mode
- Graphiques et export PDF (bientôt)
